Ага, одна из задач - запуск React Native, но не Expo-версию, а обычную
Есть два типа Expo-проектов - managed и bare. Managed - это когда библиотека Expo управляет всеми зависимостями. Второй вариант - bare, когда ты каждую нативную либу (для iOS, либо Android) подключаешь в полуручном режиме, но он более гибкий, можно использовать любые нативные либы, даже не заточенные под expo - https://docs.expo.io/introduction/managed-vs-bare/
У нас использовался вначале Managed workflow, когда его уже начало не хватать, перешли на Bare
То есть почитать нужно документацию по установке Android SDK на Windows, попробовать запустить эмулятор Android, и следующий шаг - установить React Native и запустить Hello World Application

Есть момент - это в данный момент не самая приоритетная задача, скоро будем работать над CRM-системой для ресторанов
